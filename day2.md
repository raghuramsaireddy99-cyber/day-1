# WordPress Setup Using XAMPP (Local Hosting)

## Objective
This project demonstrates how to set up WordPress locally using XAMPP, configure hosting features, customize UI/settings, and optimize performance.

# Task 1: Implement Core Concepts

## What is Hosting
Hosting is a service that allows websites to be accessible on the internet or locally.

## What is XAMPP
XAMPP is a local server package that includes Apache, MySQL, and PHP.

## Install XAMPP
1. Download XAMPP
2. Run the installer
3. Select Apache and MySQL
4. Install and open Control Panel

## Install WordPress
1. Download WordPress
2. Extract into:
   C:/xampp/htdocs/
3. Rename folder : sangeeth-com
4. Start Apache and MySQL

# Task 2: Create / Configure Features

## Create Database
1. Open http://localhost/phpmyadmin
2. Click New
3. Create database:saireddy.com

## Configure WordPress
1. Open http://localhost/saireddy.com
2. Enter:
   Database Name: saireddy.com
   Username: root
   Password: (leave blank)
3. Complete installation
